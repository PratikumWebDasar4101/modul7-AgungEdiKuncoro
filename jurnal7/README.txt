Buat lah sebuah aplikasi sederhana untuk melakukan pendataan pada mahasiswa, dengan fungsi yang dimiliki:
1. Input Mahasiswa Baru
2. Search berdasarkan nim mahasiswa dan 
3. Delete nama mahasiswa 

Ket:
Pada input data mahasiswa baru hal yang di inputkan adalah
1. Nama 
2. NIM
3. Jenis Kelamin (Radio Button)
4. Program Studi(dropbox)
5. Fakultas (dropbox)
6. Asal 
7. Moto Hidup (textarea)
Pada halaman input data mahasiswa baru terdapat menu untuk melihat data mahasiswa, setelah selesai input maka akan berpindah pada halaman lihat data mahasiswa.

Pada halaman lihat data mahasiswa terdapat menu atau fungsi untuk input data mahasiswa dan untuk melakukan pencarian data mahasiswa berdasarkan nim pada bagian atas halaman.
Pada halaman data mahasiswa yang ditampilkan adalah
1. Nama mahasiswa
2. Nim mahasiswa
3. Aksi (berisikan fungsi delete)
